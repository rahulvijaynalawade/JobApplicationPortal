# Talent Acquisition Portal

This is a full-stack web application built using Spring Boot, MySQL, React, and basic web technologies like HTML, CSS, and JavaScript. The project simulates a real-world job hiring system where job seekers and recruiters can interact on a single platform.

---

## Overview

In this system:

- Job seekers can search and apply for jobs  
- Recruiters can post jobs and manage applications  
- Admin can manage users and overall system  

The backend handles business logic and APIs, while the frontend provides a simple and user-friendly interface.

---

## Tech Stack

### Backend
- Java (Spring Boot)  
- MySQL  

### Frontend
- React.js  
- HTML  
- CSS  
- JavaScript  

### Tools
- Postman (API testing)  

---

## Features

- User registration and login with different roles (Job Seeker / Recruiter / Admin)  
- Recruiters can post, update, and delete job listings  
- Job seekers can browse and apply for jobs  
- Resume and cover letter submission  
- Basic skill management for job seekers  
- Application status tracking  
- Simple notification system  
- Admin can manage users and content  
- Responsive frontend for better user experience
